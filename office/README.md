
## Word

#### 如何设置多级标题
* "定义新的多级别表"在<多级列表>下，不在<编号>下
* “要在库中显示的级别”为2，表示1.1,1.2,1.3...;“要在库中显示的级别”为3，表示1.1.2,1.1.3...;请灵活使用
* 标题在设置多级列表时就定好对应关系（第一章->标题1，1.1->标题2）
* "正规形式编号"和“包含的”级别编号来自"不要忘记设置
